## 👋 Sobre mim

Estudante de **Sistemas de Informação na UFPE** (7º período) e desenvolvedor **Full Stack**. Atuo na intersecção entre design de interfaces modernas e arquiteturas escaláveis, com foco na stack JavaScript — **Node.js, React e MongoDB**.

Do protótipo ao deploy, construo soluções completas que combinam base acadêmica sólida com prática real em projetos. Meu objetivo: criar experiências digitais que sejam tecnicamente robustas e genuinamente focadas no usuário.
